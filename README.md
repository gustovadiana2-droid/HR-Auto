# HR-Auto — Информационная система кадрового учета и расчета заработной платы

## Описание проекта
HR-Auto — это информационная система для автоматизации документооборота и кадрового учета. Система предназначена для организаций, желающих оптимизировать работу кадровой службы и бухгалтерии.

## Функциональность
- Ведение личных дел сотрудников
- Оформление приказов о приеме, переводе и увольнении
- Учет рабочего времени (табель)
- Расчет заработной платы (начисления и удержания)
- Формирование расчетных листков в формате PDF
- Подготовка отчетности для ФНС и СФР
